# Курсовая работа Пользовательское приложение "Текстовые квесты"


## Описание задачи 
Разработать кроссплатформенное приложение «Текстовые квесты».

Для приложения разрабатывается пользовательский интерфейс, который должен быть интуитивно понятным пользователям приложения.

Пользователю предоставляется возможность прохождения выбранного текстового квеста. В главном меню есть возможность открыть окно с выбором истории. Пользователь осуществляет выбор и переходит в окно, где отображаются основной текст и две кнопки выбора. Когда квест будет завершен, пользователь может вернуться в главное меню.


## Техническое задание
##### 1. Общие требования

Разработать кроссплатформенное приложение «Квестовые тексты» на языке программирования C++ с использованием графической библиотеки Qt.

##### 2. Главное меню

Главное меню – стартовое интерфейсное окно, содержащее кнопки «Выбор истории», «Помощь» и «Выход».

После нажатия кнопки «Выбор истории» выполняется закрытие главного меню и открытие окна с выбором историй.

##### 3. Выбор истории

Интерфейсное окно выбора историй можно открыть через главное меню. Добавляются кнопки двух историй и кнопка перехода в главное меню. При нажатии на одну из кнопок историй («Traum» или «Служба поддержки молящихся») закрывается окно выбора историй и открывается окно выбранной истории. После нажатия кнопки «Главное меню» пользователь возвращается в главное меню.

##### 4. Интерфейсное окно выбранной истории

 В подкаталоге «Data» находятся заранее подготовленные текстовые документы, пронумерованные в необходимом порядке. В зависимости от выбранной истории отображается нужный текст истории и текст вариантов ответов на кнопках. Пользователь делает выбор нажатием одной из кнопки выбора. После нажатия кнопки, в зависимости от сделанного выбора, отображаются новый текст истории и текст выборов. Также добавляется кнопка «Главное меню» для выхода в главное меню.

##### 5. Панель инструментов 

Панель инструментов может содержать следующие кнопки:

1. Главное меню
2. Помощь
3. О приложении
4. Выход

##### 6. Помощь

Краткое руководство по использованию программы.

##### 7. О программе

Сведения о версии и авторе.


## Пользовательское описание
В главном меню есть возможность выйти из игры или же перейти в меню выбора историй. Чтобы выбрать историю, нужно в меню выбора истории нажать на соответствующую кнопку. Чтобы сделать выбор, следует нажать на кнопку с выбранным ответом. Иногда история не имеет вариантов ответа, поэтому, чтобы продолжить историю, нужно нажать кнопку \"далее\". Когда история будет завершена, есть возможность перейти в главное меню. Сверху, в панели инструментов, можно выйти из игры, узнать информацию о приложении или выйти в главное меню.

Как пройти историю:

1. В главном меню нажать на кнопку "Выбор истории", чтоб перейти в меню выбора истории.
2. Выбрать историю.
3. Проходить текстовый квест, выбирая варианты ответов.
4. Если вариантов нет, нажать кнопку "далее". 
4. Когда история закончится, выйти в главное меню.


## Техническое описание деталей реализации
1. Установить дополнительное ПО на компьютер:
- Qt (версия 5.14);
- Cmake; 
2. Загрузить проект из [GitHub](https://github.com/Daria2001/Course_Work_TP/tree/master).
3. Собрать проект запуском файла built.bat в подкаталоге Build.
4. В подкаталоге Debug запустить файл CourseWork.exe
